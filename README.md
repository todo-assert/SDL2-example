# SDL2-example
# 🌠SDL2 & device jpeg application 🌠

🤣PC 上调试后可以 make PC_SIMULATOR=n CC=xxx-xxx-gcc 编译设备端程序

🤣需要自己安装或者编译 libjpeg, 测试使用的为 libjpeg9b

🤣已实现自适应满屏及铺满屏幕

🤣可设置横屏或者竖屏

🤣编译后执行 ./jpeg-example < jpeg 路径 >

🤣增加 scaler 旋转 0° - 270°
